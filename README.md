# TypeScript Node Express Boilerplate

A simple, clean, and scalable **Node.js + Express + TypeScript** backend starter project.
This repository is ideal for learning backend fundamentals and for building REST APIs that can grow into production-ready services.

---

## Features

- Node.js with Express
- TypeScript for type safety
- Environment variable support
- Clean and modular folder structure
- Easy to extend and scale
